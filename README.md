#API de Análise e Monitoramento de Golpes Digitais
Este projeto apresenta uma API RESTful desenvolvida em Java com Spring Boot, criada para catalogar, monitorar e analisar incidentes de golpes digitais, como fraudes via PIX, phishing e clonagem de aplicativos de mensagens. A solução foi projetada para transformar relatos isolados de vítimas em um dataset estruturado, permitindo análises estatísticas e geração de inteligência acionável para instituições financeiras, pesquisadores e equipes de segurança digital.

objetivo do trabalho 
Criar uma base confiável de informações sobre fraudes digitais para possibilitar monitoramento, mapeamento de tendências, geração de relatórios e identificação de padrões de ataques.
